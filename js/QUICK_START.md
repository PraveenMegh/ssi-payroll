# Quick Start (5 minutes)

1) Run locally
- Open `index.html`, OR run a local server:
  - `python -m http.server 8000`

2) Login
- admin / admin123

3) Setup (Admin)
- Products → Add products + pack sizes + price
- Clients/Vendors → Add your clients
- Users → Create Stock, Dispatch and Sales users

4) Start operations
- Inventory → Opening Stock
- Inventory → Daily IN / Daily OUT
- Sales Orders → Create Draft → Save → Edit → Submit
- Dispatch → Process submitted orders (Urgent first, then FIFO)
- Reports → Monthly sales + trend
